函数与方法：

函数是一端打包的代码，它们往往会读入参数，并返还一个值。
当然有些函数没有参数，有些函数没有返还值。
尤其是当我们需要反复进行一系列操作的时候，我们就可以创建一个函数，这样每次我们需要进行
该操作的时候只需要调用该函数，而不用把代码全部重新写一遍。

学了函数以后编程就可以变得很有趣了嘿嘿嘿~

I. 声明及定义函数：
   
      我们以main function 为例：

   public static void main(String[] args){
       //do something
   }
   
      我们来依次解释一下每个词的意思：
   
   public  :  公有。package里所有类都能访问这个函数。与之对应的是私有private，即
                                         只有在这个类里才能调用这个函数。
                                         
   static  :  静态。现阶段你可以不用具体了解static，但是你应该注意：
                    1. main() 必须是静态
                    2. 在main()对应的这个类里，你定义的其他函数也应该是静态。
                    3. 如果一个类里没有main()，建议都不要用静态。不写static就行了。
                       比如：public int foo(){};
                       
   void    :  返还类型。即这个function返还的值的数据类型。这个类型可以是：
                                                  原始类型：如int, boolean等
                                                  不是原始但是java知道的类型：如String，int[],或你定义的类型。
                        void : 这个函数不返还任何值。
                        
   main    :  函数名。除了main以外你随便起名。与给变量命名一样，建议起描述性强的好记的名字。
   
   String[] args    :    函数的参数。这个参数只有在函数内部能用。
                                                      函数可以没有参数，但必须有小括号( )。
                                                      如果需要多个参数，用逗号 , 隔开，比如 foo(int x, int y)

其中：公私有，返还类型，和函数名是必须的！
                                                      
以上就是声明函数需要的关键词，除此之外，为了让你的函数正常运行，你还需要知道：

return      返还
            1. 除非函数的返还类型为void，否则一个函数必须有一个return。
            2. return的值必须与返还类型相同，比如：
               
               public int foo(){
                     //do something
                     return 0;     //must return an int
               }
             
            3. 进入函数后，java依然是从上往下读，但当读到return之后，函数将不会继续。
            
            4. 函数返还的值将会被当做本次调用的该函数的值。（不懂？看实例！）
